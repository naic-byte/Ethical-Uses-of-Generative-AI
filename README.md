Overview

This project is a static informational website designed to educate students and academics on Generative AI (GenAI), its functions, capabilities, limitations, and the critical importance of its ethical use in educational settings. The site is structured into four main pages: a landing page, an FAQ section, a detailed guide on ethical AI use, and an article presenting a relevant analogy.

The website is built to be clean, modern, fully responsive, and easy to navigate, utilizing the **Bootstrap 5** framework.

Installation / How to Run

Since this is a static website, no special installation or server setup is required.

1.  Clone or Download: Download the project folder structure.
2.  Navigate: Go to the project root directory.
3.  **Open:** Double-click `index.html` to open the website in your default web browser.

The site can be viewed locally or can be accessed throught the published link in the github.
